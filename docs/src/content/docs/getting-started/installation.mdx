## As an Agent Skill (Recommended)

### Using skills.sh

The easiest way to install msgraph is via [skills.sh](https://skills.sh):

```bash
npx skills add merill/msgraph
```

This automatically discovers the skill in the repository and installs it into your agent's skill directory.

### From GitHub Releases

Download `msgraph.zip` from the [latest release](https://github.com/merill/msgraph/releases/latest) and extract it into your agent's skill directory:

```bash
# macOS/Linux
curl -fsSL -o msgraph.zip https://github.com/merill/msgraph/releases/latest/download/msgraph.zip
unzip msgraph.zip -d ~/.claude/skills/
```

```powershell
# Windows (PowerShell)
Invoke-WebRequest -Uri "https://github.com/merill/msgraph/releases/latest/download/msgraph.zip" -OutFile msgraph.zip
Expand-Archive msgraph.zip -DestinationPath "$env:USERPROFILE\.claude\skills"
```

The skill package includes pre-built binaries for all supported platforms (macOS, Linux, Windows — amd64 and arm64). No additional downloads are needed.

## Manual Installation

### From Source

```bash
git clone https://github.com/merill/msgraph.git
cd msgraph
make build
```

The CLI is built to `./msgraph`. Copy it somewhere on your PATH:

```bash
make install  # Installs to skills/msgraph/scripts/bin/
```

## Verify Installation

```bash
msgraph --help
```

You should see the available commands: `auth`, `graph-call`, and `openapi-search`.
